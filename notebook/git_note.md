## 建立 git 版本控制


1.建立GIT版控
到terminal:
>$ mkdir 要使用的資料夾
>$ cd 要使用的資料夾
>$ git init
2.新增檔案
>$ echo HI > 某文字檔
(可以開始輸入東西)
>$ type 某文字檔
3.將 某文字檔 新增至暫存區
> git add 某文字檔 
4.觀察檔案狀態
>$ git status  
4.將 某文字檔 提交(commit)至Repository
git commit -m "add 某文字檔"  
5.push
>$ git push

參考資料來源：https://kingofamani.gitbooks.io/git-teach/content/chapter_2/
