## 建立 git 版本控制


1.建立GIT版控
到terminal:
 <p>$ mkdir 要使用的資料夾</p>
 <p>$ cd 要使用的資料夾</p>
 <p> $ git init</p>
2.將 某檔案 新增至暫存區
<p> git add 某檔案 </p>
3.觀察檔案狀態
<p> $ git status  </p>
4.將 某檔案 提交(commit)至Repository
<p>git commit -m "add 某文字檔"  </p>
5.push
<p> $ git push</p>

參考資料來源：https://kingofamani.gitbooks.io/git-teach/content/chapter_2/
